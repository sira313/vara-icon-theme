# Vara Icon Theme
![ss](https://raw.githubusercontent.com/sira313/vara-icon-theme/refs/heads/main/Screenshots/ss.png)
Vara? Gak tau artinya apaan, bingung aja kasi nama. Vara mungkin variabel, karna pake variabel `primary` dan `secondary` untuk ubah warna `*.svg`.

## Sebelum Install
Icon ini pake Qogir untuk fallback. Jadi install dulu [Qogir Icon Theme](https://github.com/vinceliuice/Qogir-icon-theme) ya...

Kenapa? karna saya suka Qogir, kalo mau ganti yang lain. Edit aja [baris ini](https://github.com/sira313/vara-icon-theme/blob/c3df1b0ad6677cd72019b02e94c549369f50f14c/index.theme#L4).

## Installasi
Clone atau download via tombol download di atas.
```
git clone https://github.com/sira313/vara-icon-theme && cd vara-icon-theme
```
Kasi akses ekse ke script
```
chmod +x generate.sh
```
Trus jalanin
```
./generate.sh
```
Copy paste warna hex yang kalian mau.
Lokasi installasi di ~/.local/share/icons/Vara
